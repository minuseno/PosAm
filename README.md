# PosAm Tests Example

###  Clone this Repository

### Run Commands:

```
npm install
```
```
npx playwright install
```
## Run Tests:
- FrontEnd Tests:

```
npm run run:fe:tests:dev
```
```
npm run run:fe:tests:prod
```
- API Test:
```
npm run run:api:tests:dev
```
```
npm run run:api:tests:prod
```
